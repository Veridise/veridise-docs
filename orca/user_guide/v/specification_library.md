---
title: Specification Library
sidebar_position: 9
---

To simplify the specification writing process, the AuditHub platform comes with a built-in specification library, which contains specifications for a number of common contract standards, such as ERC20, ERC721, and ERC3156. To test against these specifications, a user simply clicks on the `[V] Spec Library` option when selecting the `[V] Spec Source` during Job creation, chooses the appropriate contract standard, and chooses the specifications to test against.
