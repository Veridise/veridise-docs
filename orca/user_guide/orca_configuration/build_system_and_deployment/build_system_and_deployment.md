# Build System and Deployment Scripts

OrCa supports Solidity projects that use either [Foundry](https://getfoundry.sh/) or [Hardhat](https://hardhat.org/) (version 2 or 3).

While both Foundry and Hardhat projects are supported, support is most robust for Foundry, so we suggest using Foundry when possible.

In this portion of the documentation, we will cover:

1. [How to Set Up a Foundry Project for OrCa](foundry.md)
2. [How to Set Up a Hardhat Project for OrCa](hardhat.md)
3. [Debugging Build and Deployment Issues](debug.md)
