## What is ZK Vanguard?

ZK Vanguard is a static analysis tool used to discover common vulnerabilities in zero-knowledge (ZK) circuits.
ZK Vanguard performs its analysis over the [LLZK intermediate circuit representation](https://veridise.github.io/llzk-lib/main/)
By performing analysis on circuits compiled into LLZK, ZK Vanguard is able to support any ZK language that supports LLZK compilation.
Currently, ZK Vanguard is able to provide analysis for [Zirgen](https://github.com/risc0/zirgen) circuits,
with support for [circom](https://docs.circom.io/), [halo2](https://zcash.github.io/halo2/), and [Plonky3](https://polygon.technology/plonky3) coming soon.
