For security companies, AuditHub is a comprehensive audit facilitation platform. Besides the offered security tools, AuditHub allows users to collaborate while reviewing the source code of a project, and define and manage security `issues`. These issues are reported by the auditors to the project's developers from within AuditHub, which allows the two parties to further collaborate on addressing these issues. They can be created from manual review or from findings, and the final list of issues is attached to the audit report.
